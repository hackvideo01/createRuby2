# Tạo Ruby (Furigana) cho tiếng Nhật

## Cách chạy

- Mở file `index.html` bằng trình duyệt (Chrome/Safari/Edge).
- Nhập 1 câu tiếng Nhật vào ô input.
- Bấm **Tạo ruby** (hoặc nhấn `Ctrl+Enter` / `Cmd+Enter`).

## Ghi chú

- Ruby (furigana) được tạo tự động bằng `kuroshiro` + `kuromoji` tải từ CDN.
- Nếu máy không tải được CDN, trang sẽ **fallback** và chỉ hiển thị chữ thường (không ruby).
